# webdriverio tests example

## Install
1. Install Java:
    ```
    brew cask install java
    ```
2. Check:
    ```
    java -version
    ```

2. Install NodeJS:
    ```
    brew install nvm
    nvm install 10.15.0
    ```

4. Check:
    ```
    node -v
    ```
3. Install packages:
    ```
    npm install
    ```

## Run tests
Execute:
    ```
    npm run test
    ```
